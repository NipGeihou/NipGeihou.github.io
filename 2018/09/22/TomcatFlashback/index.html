Tomcat 捕获闪退问题

发表于 2018-09-22 | 更新于 2018-11-28 | 评论数：

在Tomcat的根目录下，在空白处按住Shift+右击，选择“在此处打开Powershell”（Win10以下版本为CMD），cmd、PowerShell均可。

输入startup.bat run 回车，便会提示什么问题导致闪退，根据所提示的内容重新配置环境变量。

在设置环境变量时，JAVA_HOME、JRE_HOME等根目录路径变量值时结尾无需添加”;”（分号）

如果还是不行，手动设置JDR、JER路径

tomcat/bin/startup.bat 顶部添加，shutdown.bat同理
